Extra Hard Returns Remover
Removes all hard returns, except between multiply-spaced lines or in lists
  • To write a list without losing its hard returns, keep all the elements in a given block single-spaced and use dashes, asterisks, numbers, or tabs at the start of each line after the first
  • Text enclosed in <pre></pre> tags will be preserved; each tag should be placed on its own line and will be removed
  • Lines beginning with ">", such as " > > " in email replies, will be removed along with any spaces surrounding them; "---Original Message----" will be added before the region, and "-------------" will come afterward
  • Select the text to modify, or leave it completely unhighlighted to work on the whole text
  • If necessary, undo changes by selecting Undo (Edit > Undo) as many times as necessary
